# Military Asset Management System

A full-stack web application designed for base commanders and logistics personnel to securely manage, trace, and execute logistics for military assets.

## Quick Links
- **Documentation**: Check out `Documentation.md` for full API architecture, details on RBAC rules, Data Models, and system instructions.

## Tech Stack
- **Frontend**: React (Vite) + Tailwind CSS
- **Backend**: Node.js + Express
- **Database**: SQLite
- **Security**: JWT-based Role-Based Access Control

## Requirements Fulfilled
- ✅ Full database schema tracking vehicles, weapons, and ammunition.
- ✅ API Endpoints handling intra-base array logic securely using DB Transactions.
- ✅ Strict RBAC locking down Admin vs Commander scopes.
- ✅ Dynamic frontend featuring purchases, transfers, and field assignments.
